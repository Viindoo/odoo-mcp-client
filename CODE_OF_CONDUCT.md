# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its Code of Conduct.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: we are committed to a harassment-free, welcoming, and respectful experience
for everyone, regardless of background or identity. Be considerate and constructive in
all project spaces (issues, pull requests, discussions, and any related communication).

## Reporting

If you experience or witness unacceptable behavior, report it to the project maintainers
at the Viindoo organization via https://viindoo.com/page/contact-us. All reports will be
reviewed and investigated promptly and fairly, and reporter confidentiality will be
respected.

Maintainers are responsible for clarifying and enforcing this Code of Conduct and may
take any action they deem appropriate, up to and including a temporary or permanent ban
from project spaces, in line with the Contributor Covenant's enforcement guidelines.
